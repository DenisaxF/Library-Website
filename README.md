# Library Website
A Full-Stack web application for an online bookstore, built with Node.js and MySQL.

# Features
- User Accounts: Registration and Login functionality.

- Dynamic Catalog: Books are fetched and displayed dynamically from a database.

- Modular Design: Clean code structure using custom modules and EJS templates.

- Responsive UI: Styled with CSS and SCSS for all device types.

# Tech Stack
- Backend: Node.js, Express.js

- Frontend: EJS, HTML5, CSS3/SCSS, JavaScript

- Database: PostgreSQL
